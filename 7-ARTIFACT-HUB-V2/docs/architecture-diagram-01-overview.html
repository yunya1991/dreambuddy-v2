图1：整体架构设计

用户入口层 → 统一网关 → Hub核心服务 → 共享数据层

📅 2026-05-19 📂 7-ARTIFACT-HUB-V2 🏷️ 架构设计

架构要点

  • 用户入口层：Web、小程序、API、Telegram Bot 通过 3-FRONTEND 统一接入
  • 统一接入网关：Dream Universal Gateway (:3000) 处理认证、鉴权、积分
  • Hub核心服务：7-ARTIFACT-HUB-V2 (:3456) 提供路由、工单、追踪、事件流
  • 共享数据层：MetaStore + Artifacts + Config 统一存储
  • 六部门模型：研究部/交易部/治理部/运营部/HR/市场部
  • 六人董事会：由6位部长Agent组成的治理委员会
  • 决策分级：L1部门自处 → L2董事会协调 → L3人工审批
