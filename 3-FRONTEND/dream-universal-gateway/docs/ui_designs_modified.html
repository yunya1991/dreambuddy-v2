📝 设计修改总结

✅ 修改1: 用户信息 + 积分位置调整

原设计: 用户信息（张三）+ 积分在右侧面板底部
新设计: 移到左侧栏左下角， closer to 用户操作区域
优势: 更符合用户习惯，信息获取更直观

✅ 修改2: 设置展开四大功能

原设计: 设置在左侧栏，点击后跳转到右侧面板
新设计: 点击设置后，在左侧栏内展开显示四大功能菜单
优势: 减少页面跳转，操作更流畅

✅ 修改3: 新增数据卡片模块

位置: "新建对话" 和 "API配置帮助" 之间
内容: 行情卡片 + 评分卡片 + 持仓卡片（新增）
优势: 快速查看市场数据和持仓状态

✅ 修改4: OKX API配置完整字段

原有字段: API Key + Secret Key + Passphrase
新增字段: 环境选择（Demo/Live）
优势: 更清晰地区分模拟盘和实盘
