📝 最终设计总结

✅ 左侧栏设计

顶部: Logo + 搜索 + 对话列表
中间: 新建对话 + 数据卡片（折叠）+ 设置（折叠）
左下角: 用户信息 + 积分（常驻显示）

✅ 右侧面板交互

默认: 显示分析进度 + 市场概况
点击设置: 显示四大设置功能菜单
点击具体设置: 显示该设置的详细配置
返回: 每个设置页都有"← 返回"按钮

✅ 设计优势

简洁美观: 默认折叠，减少视觉干扰
操作流畅: 无需页面跳转，右侧面板动态切换
信息清晰: 用户信息在左下角，符合使用习惯
扩展性强: 数据卡片和设置都可以无限扩展
