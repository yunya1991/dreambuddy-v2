## 交易合规

### 风险参数

| 参数 | 上限 | 说明 |
|------|------|------|
| 单币种仓位 | 30% | 账户净资产占比 |
| 总多头仓位 | 80% | 账户净资产占比 |
| 总空头仓位 | 50% | 账户净资产占比 |
| 杠杆倍数 | 5x | 最大杠杆 |
| 止损比例 | 10% | 单笔最大亏损 |
| 日亏损上限 | 5% | 账户净资产日亏损 |

### 禁止交易

| 类型 | 描述 | 违规代码 |
|------|------|----------|
| 禁止做多 | 极端利空场景 | TRADE_001 |
| 禁止做空 | 极端利多场景 | TRADE_002 |
| 禁止开仓 | Regime不明 | TRADE_003 |
| 禁止加仓 | 回撤超限 | TRADE_004 |
| 禁止全仓 | 杠杆超限 | TRADE_005 |

## 策略合规

### 策略参数边界

| 策略类型 | 参数 | 最小值 | 最大值 | 默认值 |
|----------|------|--------|--------|--------|
| 趋势跟随 | 持仓周期 | 1h | 7d | 4h |
| 网格交易 | 网格间距 | 0.5% | 5% | 1% |
| 突破策略 | 止损比例 | 1% | 10% | 3% |
| 均值回归 | 止盈比例 | 2% | 20% | 5% |

### 策略审查清单

- [ ] 策略逻辑清晰可解释
- [ ] 参数设置在合理范围内
- [ ] 风险敞口可接受
- [ ] 回测结果满足要求
- [ ] 无已知风险事件

## 执行合规

### 下单频率

| 时间窗口 | 最大订单数 | 说明 |
|----------|------------|------|
| 1分钟 | 10 | 防止过度交易 |
| 5分钟 | 30 | 短期限制 |
| 1小时 | 100 | 中期限制 |
| 24小时 | 500 | 日内限制 |

### 仓位限制

- 单笔订单最大金额: $10,000
- 24小时最大交易额: $100,000
- 最大持仓品种数: 5

## 文档合规

### 必需记录

| 记录类型 | 保存时间 | 说明 |
|----------|----------|------|
| 交易记录 | 5年 | 完整交易历史 |
| 决策记录 | 3年 | 决策链路 |
| 审计日志 | 1年 | 系统日志 |
| 风险报告 | 3年 | 风险事件 |

### 追溯要求

- 每笔交易需关联决策依据
- 异常交易需说明原因
- 系统变更需记录时间线

## 审查流程

### 审查层级

```
┌─────────────────────────────────────┐
│          Level 1: 自动审查           │
│  系统自动检查参数/频率/仓位           │
└─────────────────┬───────────────────┘
                  │
                  ▼
        ┌─────────────────┐
        │  Level 2: AI审查  │
        │  ai-trading-    │
        │  compliance      │
        └────────┬────────┘
                 │
                 ▼
       ┌─────────────────┐
       │ Level 3: 人工审查 │
       │ 治理委员会        │
       └────────┬────────┘
                │
                ▼
       ┌─────────────────┐
       │   审查结果输出    │
       │ PASS / WARN / FAIL│
       └─────────────────┘
```

### 审查时间

| 审查类型 | 时间要求 | 说明 |
|----------|----------|------|
| 实时审查 | <100ms | 自动执行 |
| 策略审查 | <5min | 变更申请 |
| 人工审查 | <24h | 复杂问题 |
