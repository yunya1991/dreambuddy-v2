### L1 分层执行机制（核心优势）

```
【阶段1: 加仓期（三层未满）】
  止损策略:
    • 止损仅作为【硬性保护】
    • 阈值放宽至极端情况（跌破Level 3极限价）
    • 目的: 让三层加仓有足够空间完成

【阶段2: 持仓期（三层加满后）】
  止损策略:
    • 止损正式启用
    • 止盈止损基于加仓后均价计算
```

### 止盈动态调整规则

| 状态 | 止盈调整 | 说明 |
|------|----------|------|
| 常规行情 | ❌ 不调整 | 保持原有止盈设置 |
| 趋势加强期 | ✅ 提高20-50% | 检测到趋势加强信号时 |
| 强信号 | ⚡ 立即止盈 | 极端行情快速离场 |

> **相比传统马丁策略的核心优势**: 传统马丁无法实现加仓与止损的动态平衡，本系统通过L1分层机制完美解决这一矛盾。
