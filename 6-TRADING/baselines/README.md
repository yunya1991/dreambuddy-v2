| 指标 | v1 BTC | v2 BTC | v1 SOL | v2 SOL | v1 ETH | v2 ETH |

**v4.0 改进点**：
1. `detect_market_regime()` — 5档形态识别，CONSOLIDATION/WEAK状态下仓位×0.5-0.75
2. TP 倍数：ATR×2/3/5 → **ATR×0.8/1.5/2.5**（TP 触发从 1 次增至 5 次）
3. 熊市强制做空：weekly close < EMA50×0.95 → 强制 SHORT（本测试期未激活，EMA50 数据不足）
4. TP 分批比例修正：30%/30%/40% → 50%/30%/20%

## 关键诊断: 熊市测试周期

**回测周期 (2025-11-27~2026-05-26) 为熊市**：BTC -15.81% / SOL -40.14% / ETH -30.25% 买持

**v0 表现优秀**：高频双向 + 做空捕获下跌（SOL +68.68%）  
**v2 核心成就**：MaxDD 大幅压缩（ETH 32.91%→7.72%），TP 开始触发  
**仍存在的问题**：熊市期 Screen1 仍返回 LONG（EMA50 数据不足 50 周，强制做空条件未激活）

## 升版门槛（v3 需超越 v0 基线）

新版本要超过 v0 基线以下至少 **2 项**：

| 指标 | BTC v0 | SOL v0 | ETH v0 | 说明 |
|------|--------|--------|--------|------|
| Sharpe | 0.715 | 3.022 | -0.336 | 必须提升或持平 |
| Max Drawdown | 18.25% | 17.60% | 32.91% | 必须降低 |
| Total Return | +8.78% | +68.68% | -8.67% | 参考 |
| Profit Factor | 1.047 | 1.248 | 0.961 | 必须≥1.0 |

## v3 优化方向

| 优先级 | 改进项 | 预期效果 |
|--------|--------|---------|
| P1 | Screen1 改用 EMA20 跨越 EMA50（而非绝对水平） | 触发 SHORT 覆盖 |
| P2 | 周线趋势连续确认（连续 N 周下跌） | 减少假信号 |
