## 关键诊断: v1 在熊市中失效

**回测周期 (2025-11-27~2026-05-26) 为熊市周期**：
- BTC: -15.81% 买持
- SOL: **-40.14%** 买持  
- ETH: -30.25% 买持

**v0 表现优秀**：高频双向交易 + 做空捕获下跌（SOL +68.68%）  
**v1 核心问题**：Screen1 周线评分未能识别熊市，持续返回 LONG 方向，马丁策略在下跌趋势中连续加仓止损

**v1 的优势**：
- MaxDD 更低（11.7% vs 18.25%，回撤控制有效）
- 马丁链完整执行（3/4 笔交易 Level3 加满）
- 不依赖高频信号（适合实盘低频操作）
