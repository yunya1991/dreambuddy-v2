## 🔗 中台驾驶舱三大闭环

### 闭环1: 交易决策闭环
```
A1(调研) → A2(分析) → A3(推演) → A4(验证) → A5(执行)
    ↑                                              │
    └──────────────── A7(门禁) ← A6(监控) ←───────┘
```

### 闭环2: 学习进化闭环
```
Episode记录 → Lesson蒸馏 → Proposal生成 → 验证门禁 → 实施 → A8复盘
```

### 闭环3: 治理合规闭环
```
合规审查 → 影子验证 → 执行监控 → 审计归因 → 持续改进
```

## 🎯 三叉戟决策模型

| 信号 | 条件 | 行动 |
|------|------|------|
| BUY | 评分≥35 + Edge>0 + 阻力突破 | 做多 |
| SHORT | 评分≤15 + Edge<-20 + 阻力确认 | 做空 |
| SKIP | 其他情况 | 空仓等待 |
