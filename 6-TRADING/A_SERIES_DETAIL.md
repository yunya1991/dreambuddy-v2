## 📊 A0-A9全流程

```
用户输入 → A0矛盾识别 → A1深度调研 → A2第一性原理 → A3沙盘推演
    → A4战术验证 → A5决策执行 → A6情报监控 → A7实践门禁 → A8知行验证 → A9离场
```

---

## A0 - 矛盾分析

**SKILL**: `dream-contradiction-theory`

**核心原理**:
- 毛泽东《矛盾论》
- 《孙子兵法》
- 克劳塞维茨《战争论》

**输出**: 市场主要矛盾清单

**SKILL**: `dream-strategy-research`

## A2 - 第一性原理

**SKILL**: `dream-first-principles`

**核心原理**:
- 市场总是沿着阻力最小方向运行
- 趋势具有延续性

**方法论**: 矛盾论 + 双维度分析

**输出**: 市场状态判断 + 阻力分析

## A4 - 战术验证
