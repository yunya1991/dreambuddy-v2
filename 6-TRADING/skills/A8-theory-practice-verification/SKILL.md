### 1. 检验（调查 A0-A7 的理论和实践结合）

## 六、A8 纯粹内部批评自循环架构（v1.5）

### 6.1 A8内部批评轨（完整自循环）

**职能**：A8独立检验A0-A7的理论与实践结合情况，生成改进建议，通过自我验证推动理论进化。

**完整自循环流程**：
```
A0/A7输出
    ↓
A8独立批评分析（纯内部，不参考任何外部视角）
    ↓
A8自评可信度 + 独立验证加成
    ↓
综合可信度 ≥ 70% → 进入观察期
综合可信度 < 70% → 暂不采纳
    ↓
观察期验证（模拟盘）
    ↓
效果评估（P&L/知行合一/认知偏差/执行速度）
    ↓
┌───┴───┐
正向        负向/持平
↓           ↓
正式采纳    触发回滚
（A0/A7升级）   （恢复改进前版本）
    ↓           ↓
绩效考核    恢复改进前版本
    ↓           ↓
更新A8      更新A8
可信度基线   绩效考核
    ↓
写入MEMORY.md长期记忆
    ↓
┌───┘
（完整自循环：批评→验证→采纳/回滚→记忆→下一轮批评）
```

**纯粹内部性保证**：
- A8**不读取**任何外部批判报告（做梦部等）
- A8**不参考**任何外部评分或建议
- A8**不考虑**"外部视角会怎么想"
- A8所有批评、验证、进化都在内部完成

### 7.1 批评可信度评估（采纳前置条件）

A8作为独立批判视角，其批评报告通过**自我评估 + 独立验证加成**判断可信度：

```
A8批评报告输出
    ↓
┌─────────────────────────────────────────────┐
│         第一阶段：A8独立可信度评估           │
│  1. A8自评可信度（0-100%）                │
│  2. 独立验证加成（覆盖度/一致性/证据）    │
│  3. 综合可信度 = A8自评 + 独立验证加成    │
│  4. 综合可信度 ≥ 70% → 进入观察期        │
│     综合可信度 < 70% → 暂不采纳，记录待查  │
└─────────────────────────────────────────────┘
```

**独立验证加成规则**：

| 验证维度 | 加成 | 判定标准 |
|:---|:---:|:---|
| **覆盖度** | +10% | A8检查了≥80%的理论实践结合点 |
| **逻辑一致性** | +10% | A8批评内部无自相矛盾 |
| **证据充分性** | +10% | A8每个批评都有具体证据支持 |
| **认知偏差识别** | +5% | A8识别到≥3个认知偏差 |
| **改进建议可操作性** | +5% | A8建议包含具体修改方案 |

**综合可信度公式**：
```
综合可信度 = A8自评可信度 + 独立验证加成（最高+40%）
边界约束：综合可信度最高100%，最低0%
```

**A8自评缺失处理**：
- 综合可信度 = 独立验证加成 × 0.8（降级20%）
- 记录: "A8自评缺失，仅用独立验证加成"
- 触发: A8必须重新执行批评（补充自评）

### 7.3 A8绩效考核机制

A8作为"批评者"，其批评质量必须被全程量化考核：

#### 考核指标

| 指标 | 计算方式 | 目标值 |
|---|---|---|
| **批评准确率** | 观察期结束后效果正向次数 / 总采纳次数 | ≥ 70% |
| **P&L改善率** | (采纳后P&L - 采纳前P&L) / 采纳前P&L | ≥ 5% |
| **认知偏差减少率** | (采纳前偏差数 - 采纳后偏差数) / 采纳前偏差数 | ≥ 30% |
| **知行合一提升** | 采纳后知行合一评分 - 采纳前评分 | ≥ 15分 |

#### 考核周期

```
每日：记录P&L变化 + 市场反馈
  ↓
每周：计算本周批评准确率 + 偏差减少率
  ↓
每月：更新A8可信度baseline + 综合评级
  ↓
每季度：调整A8批评权重 + 回顾进化效果
```

### 7.4 回滚机制（2026-04-26 v1.4更新）

#### 触发回滚的条件（每日检查**所有**条件）

> **v1.4核心改进**：回滚仅基于客观指标（P&L/知行合一/认知偏差/执行速度），删除所有主观触发条件，避免A8迎合失真

| 触发条件 | 阈值 | 检查频率 | 回滚速度 |
|---|---|---|---|
| P&L回撤 | 观察期内模拟盘亏损 > 3% | 每日 | 立即回滚 |
| 知行合一评分下降 | 采纳后评分下降 > 15分 | 每日 | 24小时内回滚 |
| 认知偏差增加 | 采纳后偏差数反而增加 | 每日 | 立即回滚 |
| **执行速度下降** | 改进后执行延迟 > 50% | 每日 | 48小时内回滚 |

**执行速度下降判定**：
- 改进前平均执行时间：T0
- 改进后平均执行时间：T1
- 如果 T1 > T0 × 1.5 → 执行速度下降 > 50% → 触发回滚
