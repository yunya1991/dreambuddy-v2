## 一、核心理论：弗洛伊德《梦的解析》五大机制

### 1.1 梦的本质

> "梦是通往潜意识的皇家大道。" — 弗洛伊德

- **清醒意识** = 显性交易逻辑（6维评分、门禁、止损）
- **被压抑物** = 系统不敢表达但隐隐感知的判断
- **梦的伪装** = 市场叙事表象 vs 底层真实驱动力的错位

### 1.2 五种"梦的工作"机制

| 机制 | 原意 | 系统映射 | 检测方法 |
|:---|:---|:---|:---|
| **凝缩** | 多意念压缩为一 | 6维→1维丢失张力 | 维度间分歧>3分时触发 |
| **移置** | 情感转移至无关对象 | 恐惧被移置为"流动性不足" | 连续引用同一原因3+次 |
| **象征** | 抽象以具体形象呈现 | FGI/ETF作为象征代替本质 | 指标与价格背离时触发 |
| **二次修正** | 碎片编织为"合理"叙事 | Episode事后归因编织因果 | 检查归因一致性 |
| **投射** | 内在冲突投射到外部 | "市场在等催化剂"=系统无信念 | 外部归因>80%时触发 |

### 1.3 晚期发现：强迫性重复

> 不是满足愿望，而是试图掌控无法掌控的创伤。

- **检测**: 连续3+次SKIP且引用相同原因 = 创伤信号
- **干预**: 触发梦境分析，而非继续执行相同逻辑

## 二、四大功能模块

### 模块1：梦境解析器（Dream Interpreter）

**功能**: 分析Episode和决策数据中的"潜在内容"

```
输入: 最近24h Episode数据
处理:
  1. 凝缩还原: 从总分还原6维矛盾结构
  2. 移置追踪: 追踪恐惧/欲望的真实对象
  3. 象征解码: 将FGI/ETF等"象征"还原为底层驱动
  4. 二次修正剥离: 从事后归因中剥离叙事编织
  5. 投射还原: 将外部归因还原为内部能力缺陷
输出: 梦境解析报告（显性→潜在内容映射）
```

### 模块2：潜意识探测器（Subconscious Probe）

**功能**: 提取系统"想说但没说"的判断

```
输入: 被门禁拦截的决策、被评分压制的维度
处理:
  1. 反事实推演: 如果门禁不存在，系统会怎么操作？
  2. 被压制维度提取: 哪个维度的信号被否决了？为什么？
  3. 潜台词挖掘: 连续SKIP的真正原因是什么？
输出: 潜意识快照（被压制的判断清单）
```

## 四、核心原则

1. **梦不受门禁约束** — 做梦部的输出不需要通过Pretrade Gate，它不是交易决策，而是决策的"潜意识层"
2. **梦允许错误** — 大胆观点的可能性可以低至30%，关键是提供不同视角
3. **梦不替代清醒** — 做梦部的输出是"参考信号"，不是"执行信号"
4. **梦需要解析** — 原始梦境不可直接执行，需要次日"清醒时刻"的解析和验证
5. **连续SKIP触发做梦** — 强迫性重复=创伤信号，需要梦境分析
6. **梦记录被压制的判断** — 系统想说但没说的，是比说了的更有价值的信息
7. **噩梦是健康的** — 定期的噩梦（黑天鹅模拟）比永远乐观更安全
