# 中台设计文档

## 🏛️ 产物中台 (Artifact Hub)

### 核心定位
内部监控和交付物流转中心，统一管理A系列报告、交易决策、产物投递。

### 架构设计

```
                    ┌─────────────────┐
                    │   用户请求入口    │
                    └────────┬────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
              ▼              ▼              ▼
        ┌──────────┐   ┌──────────┐   ┌──────────┐
        │ A1-A5    │   │ 调研部   │   │ 归档     │
        │ 专属通道 │   │ 邮箱     │   │ 处理     │
        │ (P0)    │   │(P1/P2)  │   │ (P3)    │
        └────┬─────┘   └────┬─────┘   └────┬─────┘
             │              │              │
             └──────────────┼──────────────┘
                            │
                    ┌───────▼───────┐
                    │  artifact-    │
                    │ alignment-   │
                    │ manager      │
                    └──────────────┘
```

### 路由规则
| 优先级 | 类型 | 路由目标 | 说明 |
|--------|------|----------|------|
| P0 | 紧急/关键 | A1-A5专属通道 | 直接执行 |
| P1 | 重要 | 调研部邮箱 | 48h内处理 |
| P2 | 一般 | 调研部邮箱 | 7天内处理 |
| P3 | 低 | 归档 | 批量处理 |

### 核心SKILL
- `artifact-alignment-manager` - 产物投递管理
- `auto-repair` - 系统健康自动修复
